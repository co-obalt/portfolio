# 🌐 Sohaib's 3D Developer Portfolio

A fully animated, interactive, 3D developer portfolio built using React, GSAP, Three.js, and TailwindCSS.

## 🚀 Tech Stack

- **React (Vite)** - Fast development server and production bundling
- **Tailwind CSS** - Utility-first styling for UI components
- **GSAP** - Scroll-based animations and transition logic
- **Three.js & React Three Fiber** - 3D scenes and helper elements

## 📁 Project Structure

- `src/sections/` - Major page sections (Hero, About, Services, Works, Contact)
- `src/components/` - Reusable UI and 3D components (Planet, Marquee, etc.)
- `src/constants/` - Configuration data for projects, services, and social links

## 📦 Setup & Installation

To run this project locally, clone the repository and execute the following commands:

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## 🛠️ Customization

- **Personal Info & Socials**: Modify `src/constants/index.js`
- **Styles & Custom Fonts**: Check `src/index.css`
